# Time Squats — Brand

## Positioning

Premium utilitarian app blocker. Squat to unlock distracting apps.
Female-leaning без wellness-претензий. Tool, не lifestyle.

**One-liner**: Move first. Then scroll.

## Audience

Primary: women 20–35, iPhone, light fitness, устали от doom-scrolling.
Secondary: men 20–35 с тем же паттерном (не отталкиваем).

Не GymTok, не pilates-girlies, не Goggins. Просто люди, которым нужен tool.

## Voice

**Tone**: dry / deadpan. Констатация фактов без коучинга и без агрессии.
Не друг, не тренер, не наставник. **Инструмент, который называет вещи.**

**Reference**: Things 3, Bear, Stoic, Co-Star (без mystic-части), Aesop product descriptions.

**Принципы:**
- Короткие предложения. Точка вместо восклицательного знака.
- Никогда не подбадривает («you got this», «keep going»).
- Никогда не унижает («weak», «excuses», «again?»).
- Просто констатирует: что есть, что произошло, что нужно.
- Lowercase в marketing copy. Sentence case в UI. Никогда ALL CAPS, никогда Title Case Everywhere.
- Block screen всегда minimal — «Instagram. Locked.» без оценок.
- Numbers are first-class. Counts and durations carry the meaning, not adjectives.
- No emoji. Не в маркетинге, не в UI, не в receipts. Бренд исключает emoji полностью.

**Слова, которых нет в бренде:**
- Goggins: crush, grind, discipline, warrior, no excuses, hustle
- Wellness: mindful, nervous system, ritual, sacred, journey, healing
- Coach: you got this, let's go, believe, your best self
- SaaS: boost, unlock your potential, achieve, empower

**Tone-примеры:**
- ✅ «5 squats. Then Instagram.»
- ✅ «Earned 12 minutes.»
- ✅ «247 reps this week.»
- ✅ «Streak broken. Start again.»
- ✅ «Below parallel ✓»
- ✅ «Personal best: 23 in one set.»
- ❌ «Crush your screen time goals!»
- ❌ «Reset your nervous system through movement.»
- ❌ «You got this! 💪»

## Vocabulary

Sport / measurement, не wellness / coaching. Числа важнее слов.

**Используем:**

| Категория | Слова |
|---|---|
| Action | Reps, Sets, Move, Squat, Log, Count |
| Measurement | Volume, Total, Today, This week, Average |
| Quality | Below parallel, Form, Tempo, Depth, Pace |
| Achievement | PR, Personal best, Streak, Active streak, Best day |
| State | Locked, Earned, Logged, Counted, Active, Rest |
| Time | Minutes, Earned time, Day 12, Week 47 |

**Заменяем:**

| Вместо | Используем |
|---|---|
| squats (в data context) | reps |
| good job / done | logged / counted |
| goal | target |
| streak | active streak |
| time (общее) | earned time |
| exercise | move / squat |
| start (на workout) | begin set |
| success / complete | counted |

**Не используем:**
- Journey, practice, ritual, mindful, intention, presence
- Crush, grind, push, dominate, beast mode
- Empower, transform, unlock potential, achieve
- Diminutives: squatsies, mini-move, lil' workout

**В одной фразе**: говорите как Strava recap, не как Headspace prompt.

## Sport DNA

Editorial premium на cream-палитре сам по себе читается как journal / wellness app. Sport DNA — набор продуктовых элементов, которые возвращают athletic identity, не ломая editorial tone.

**Positioning**: editorial sport / quiet athleticism — premium athletic utility, не gym-bro tool и не wellness companion.

**Reference brands:**

| Бренд | Что брать |
|---|---|
| **Tracksmith** | Premium running с literary editorial эстетикой. Главный mood. |
| **Bandier** | Athletic wear с premium retail подачей |
| **District Vision** | Editorial photography + sport |
| **Apple Fitness+ marketing** | Body movement в editorial layout |
| **Strava year-in-sport** | Sport data в premium editorial. Receipt-механика. |
| **Whoop** | Data-dense UI с premium feel |

**Продуктовые элементы Sport DNA:**

1. **Mono-counter как hero** — большое число в mono занимает 30–40% экрана. Home, Workout, Block, Stats, Receipt. Цифра больше любого текста.

2. **Pose skeleton overlay** — wireframe скелета на camera screen. Главный технический showcase + killer-screenshot для App Store + demo для TikTok.

3. **Live form feedback** — subtle подписи в mono без коучинга: «below parallel ✓», «deeper», «slower», «hold». Differentiator от всех конкурентов (никто не считает только valid reps).

4. **Body line-art** — минималистичные silhouette фигуры в squat-позиции, single-stroke. Onboarding, empty states, App Store screenshots. НЕ muscle anatomy, НЕ гантели, НЕ stock-фото.

5. **Data viz в editorial-стиле** — weekly volume bars, streak calendar (GitHub-grid style), most-resisted apps. Тонкие линии, mono для цифр. НЕ pie charts, НЕ gradients.

6. **Sound + haptics** — тонкий tick + light haptic на каждый засчитанный squat. Distinctive done-tone на завершение сета. Создаёт sport-feel вместо timer-feel.

7. **Numbers everywhere** — на любом экране, где можно показать число, показывайте. «5 squats per minute earned» вместо «squat to unlock». Числа = sport. Слова без чисел = wellness.

**Приоритет внедрения:**
1. Mono-counter hero (Home + Block + Workout)
2. Pose skeleton overlay
3. Sport vocabulary в copy
4. Weekly volume chart на Stats
5. Body line-art
6. Sound + haptics
7. Live form feedback

Первые три дают ~80% эффекта.

**Что НЕ делать:**
- Не возвращать neon / dark / lime
- Не добавлять stock-фото фитнес-моделей
- Не делать muscle anatomy / heart rate / calories — Time Squats блокировщик с упражнением, не fitness tracker

## Content pillars

1. **State, not cheer** — описываем что произошло (`Earned 12 minutes.`), не хвалим.
2. **Numbers carry meaning** — `247 squats this week.` сильнее, чем `You crushed it!`.
3. **Friction beats willpower** — продукт даёт физический шаг между импульсом и приложением.
4. **Quiet streak** — streak есть, но без gamification-celebration.

## Hook library  *(паттерны хуков для каруселей)*

- **Stat-first**: `247 reps this week.` / `4h 23min not scrolled.` / `Active streak: 12 days.`
- **Block-screen line**: `Instagram. Locked.` / `5 squats = 5 minutes.`
- **State sentence**: `Earned 12 minutes.` / `Personal best: 23 in one set.`
- **Lowercase marketing line**: `move first. then scroll.` / `pick apps. set squats. move.`
- **Empty-state line** (state the absence): `No apps blocked yet.`

## Don'ts  *(brand-level only)*

- Не Goggins, не military, не «no excuses», не «discipline»
- Не «mindful», не «sacred», не «ritual», не «journey», не «reset»
- Не celebration-language («amazing!», «you crushed it!»)
- Не wellness companion
- Не cute / playful / mascot (пока)
- Не gym-bro, не bodybuilding, не fitness tracker
- Не generic SaaS productivity tone
- Не coaching tone — продукт не подбадривает, он сообщает state
- Не emoji, нигде

## Differentiation

**vs Clearspace / PushUp Time / PushScroll**: warm editorial visual вместо clean SaaS / dark gym. Squats only — focused product.

**vs Finch / Roots**: tool, не companion. Без mascot, без журнала, без gamification beyond streak.

**vs Opal / Brick**: дешевле, доступнее, с физическим действием как unlock-механикой.

## Hook  *(основной маркетинговый канал)*

**Weekly Receipt** — shareable card каждый понедельник.
Squats count + earned time + active streak + most-resisted app.
Кнопка `Share to Stories` → image в формате 9:16.

Это единственный встроенный маркетинговый канал в продукте. Остальное — ASO + occasional posts.
