# Time Squats — Brand

## Positioning

Premium utilitarian app blocker. Squat to unlock distracting apps.
Female-leaning без wellness-претензий. Tool, не lifestyle.

**One-liner**: Move first. Then scroll.

## Audience

Primary: women 20–35, iPhone, light fitness, устали от doom-scrolling.
Secondary: men 20–35 с тем же паттерном (не отталкиваем).

Не GymTok, не pilates-girlies, не Goggins. Просто люди, которым нужен tool.

## Voice

- Дружелюбный, спокойный, без коучинга
- Без «crush», «grind», «discipline», «warrior»
- Без «mindful», «nervous system», «ritual», «sacred»
- Короткие предложения. Lowercase в маркетинге, sentence case в UI

**Примеры:**
- ✅ «5 squats. Then Instagram.»
- ✅ «Earned 12 minutes.»
- ✅ «You moved 247 times this week.»
- ❌ «Crush your screen time goals!»
- ❌ «Reset your nervous system through movement.»

## Colors (light mode)

**Общая идея**: тёплая light-палитра, которая ощущается как утренний свет в комнате, а не как стерильный SaaS. Не белая, не серая — кремовая. Бренд должен выглядеть как качественная керамика или льняная ткань, не как iOS system colors.

**Reference mood**: Aesop store, Apple Notes на бумажной обложке, Aritzia / Toteme сайты, пенный flat white в кафе. НЕ Headspace, НЕ Calm, НЕ Notion.

**Палитра по ролям:**

- **Background** — тёплый off-white с лёгким бежевым подтоном. Не cold white (#FFFFFF слишком clinical), не yellow-cream (выглядит дёшево). Цель: ощущение natural paper, чтобы экран не «светил» в глаза вечером.
- **Surface** — на тон темнее background, тот же тёплый подтон. Используется для карточек и secondary блоков. Должна быть видна без border, но без visual weight.
- **Border / divider** — ещё на тон темнее surface. Тонкие линии, не shadow. Используются вместо elevation.
- **Text primary** — НЕ pure black. Тёмно-коричневый или very dark warm grey. Цель: ощущение чернил на бумаге, не digital.
- **Text secondary** — средний warm grey-brown. Для подписей, hints, метаданных. Достаточный контраст для accessibility (минимум AA).
- **Accent / primary action** — тёплый оранжевый-coral. Не red (агрессивно), не yellow (несерьёзно), не pink (cute), не peach (cliché). Цель: дружелюбная теплота с лёгким urgency. Используется для кнопок, активного счётчика, главных CTA.
- **Success** — приглушённый sage / olive green. НЕ bright lime, НЕ kelly green. Цель: спокойная завершённость, без celebration-vibe.
- **Alert / locked state** — насыщенный terracotta-red. НЕ scarlet, НЕ ярко-красный. Цель: «остановись», но без паники.

**Правила:**
- Никаких градиентов. Только сплошные заливки.
- Никаких blur-shadow. Глубина — через 1px borders и контраст surface levels.
- Контраст текста проверять по WCAG AA минимум.
- Accent используется sparingly: 1 раз на экран как фокус, не как декор.
- Dark mode опционально на v2, не приоритет.

## Typography

**Общая идея**: пара из двух шрифтов с противоположным характером, и опционально mono для цифр. Теплота serif balances precision sans. Не один универсальный шрифт.

**Reference mood**: editorial magazine layout (свежий fashion / wellness журнал, но не Vogue), product pages Aesop, app interface Stoic / Things 3. НЕ tech startup landing page, НЕ Material Design, НЕ Apple HIG default.

**Типографические роли:**

- **Display (hero, заголовки экранов, large numbers)** — **современный serif с характером**. Должен ощущаться как редакционный, человеческий, premium. Не классический book-serif (старо), не slab-serif (слишком статично), не decorative (slишком много personality). Цель: warmth + intelligence. Используется в умеренной weight (Regular / Medium), не Bold — чтобы оставить дыхание.

- **Body / UI (всё в интерфейсе — кнопки, лейблы, описания)** — **нейтральный premium sans-serif**. Чистый, читаемый, без характера. НЕ Outfit / Poppins / DM Sans (rounded friendly = generic SaaS). НЕ humanist sans (Lato / Open Sans = corporate). Цель: невидимый шрифт, который не спорит с serif заголовком. Optical sizes полезны.

- **Counter / timer / stats (squat count, time, numerical data)** — **monospace**. Не для всего интерфейса, только для чисел, которые меняются (counter, таймер, stats). Цель: tabular alignment + ощущение «измерения», precision. Modern geometric mono предпочтительнее, чем typewriter-style.

**Принципы pairing:**
- Serif Display + Sans Body — главная пара, создаёт editorial feel
- Mono — третий голос только для данных, не для текста
- Все три шрифта должны хорошо работать в light mode на тёплом фоне
- Все три должны иметь open license или affordable lifetime license

**Размерная шкала** (modular, не arbitrary):
- Hero / display: very large (для одного слова или числа на экране)
- Section heading: large
- Subheading: medium
- Body: standard (16px equivalent)
- Caption: small
- Micro: extra small (для меток, timestamp)

**Line-height**: щедрый для body (читаемость), компактный для display (impact). Никаких squeezed lines.

**Weight usage**: предпочтительно Regular + Medium, иногда Semibold для emphasis. Bold избегать — слишком aggressive для бренда. Italic — только в serif display, очень sparingly.

**Casing:**
- UI labels: sentence case
- Buttons: sentence case
- Marketing / hero copy: lowercase допустим как стилистический выбор
- НЕ ALL CAPS, НЕ Title Case Everywhere

## Icon

Минималистичная буква **T** или **TS** в круге / квадрате с скруглёнными углами.
- Фон иконки: accent (тёплый coral из палитры)
- Литера: cream background-tone из палитры
- Шрифт литеры: тот же serif display, что в приложении (не сторонний)
- Без gradient, без emoji, без персика, без mascot
- Без skeuomorphism, без glass effect, без 3D

Альтернатива: абстрактный геометрический symbol — две arc-линии (верхняя и нижняя позиция squat), single-stroke, той же логикой цветов.

Цель: иконка на home screen должна выглядеть как premium utility (Bear, Things, Day One), не как fitness app, не как wellness app, не как productivity SaaS.

## Visual rules

- Light mode по умолчанию. Dark mode — опционально, не приоритет
- Photography: natural light, минимализм, никаких stock-fitness
- Illustrations: line-art, single-stroke, monochrome accent
- Никаких neon, gradient mesh, glassmorphism, brutalism, Y2K
- Whitespace важнее декора

## Copy patterns

**App Store:**
- Title: `Time Squats: Block & Move`
- Subtitle: `Squat to unlock TikTok & IG`

**Onboarding:**
- «Pick apps to block.»
- «Set squats per unlock.»
- «Move when it's time.»

**Block screen:**
- «Instagram is locked.»
- «5 squats to unlock 5 minutes.»
- Button: `Start`

**Receipt (weekly share):**
- «247 squats this week.»
- «4h 23min not scrolled.»
- «Streak: 12 days.»

## Don'ts

- Не lime, не toxic green, не neon
- Не pink, не cursive, не персик
- Не Goggins, не military, не «no excuses»
- Не «mindful», не «sacred», не «ritual»
- Не Outfit / Poppins / DM Sans / cute rounded geometric sans
- Не bubbly / playful / tech-startup типографика
- Не mascot (пока)

## Differentiation

**vs Clearspace / PushUp Time / PushScroll**: warm light visual вместо clean SaaS / dark gym. Squats only — focused product.

**vs Finch / Roots**: tool, не companion. Без mascot, без журнала, без gamification beyond streak.

**vs Opal / Brick**: дешевле, доступнее, с физическим действием как unlock-механикой.

## Hook

**Weekly Receipt** — shareable card каждый понедельник.
Squats count + time saved + streak + most-resisted app.
Кнопка `Share to Stories` → image в формате 9:16.

Это единственный маркетинговый канал в продукте. Остальное — ASO + occasional posts.
